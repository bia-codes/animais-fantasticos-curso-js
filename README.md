🚀 **Projeto Animais Fantásticos**

 Este projeto é o resultado prático da meta de Aprendizado Aplicado e Compartilhado do Q2, focado na aplicação dos conceitos aprendidos no curso JavaScript Completo ES6 da Origamid. Este projeto foi desenvolvido durante o curso de JavaScript Completo ES6+ da Origamid. O objetivo principal foi aplicar na prática conceitos modernos de JavaScript, desde a manipulação do DOM até a organização de código, além da refatoração completa da aplicação web já existente.
 


🛠️ **Tecnologias e Ferramentas**

 Linguagem: JavaScript



🚀 **Como Executar o Projeto**

1. Criar a pasta do projeto
   
   Escolha um local no seu computador, clique com o botão direito e crie uma nova pasta (ex: animais-fantasticos).

3. Abrir o terminal dentro da pasta
   
   Windows: Entre na pasta, clique na barra de endereços no topo, digite cmd e dê Enter.
   
   Mac/Linux: Clique com o botão direito na pasta e selecione "Abrir no Terminal".

3. Clonar o repositório
   
   Copie e cole esse comando no terminal:
   
       git clone https://github.com/bia-codes/animais-fantasticos-curso-js.git

5. Visualizar o código
   
   Abra a pasta com o VS Code para visualizar o código.

6. Instalação de dependências
   
  No terminal do VS Code, digite o comando npm install, para instalar as dependências, e em seguida o comando npm run dev para buildar o projeto.
